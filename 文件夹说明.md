# 文件夹说明

## 常用入口

- `点我启动软件.command`：在这台 Mac 上双击启动软件。
- `开启远程试用.command`：在这台 Mac 上生成临时网址，发给别人远程试用。
- `Windows一键打包.bat`：复制到 Windows 电脑后，双击生成 Windows 免安装交付包。

## 核心程序

- `app.py`：后端主程序。
- `run_desktop.py`：打包 exe/app 时使用的桌面启动入口。
- `requirements.txt`：Python 依赖列表。
- `templates/`：网页模板。
- `static/`：前端页面、样式、脚本。

## 数据和附件

- `data/app.db`：系统数据库，最重要。
- `uploads/`：上传的附件。
- `backups/`：数据库和模板备份。

## Excel 模板

- `管理系统.xlsx`：原始模板。
- `管理系统_五行示例数据.xlsx`：带五行示例的总模板。
- `导入示例Excel/`：每个模块单独的导入模板。

## 文档

- `README.md`：系统总说明。
- `docs/`：打包、交付、Windows 用户说明。

## 工具

- `scripts/`：生成示例 Excel 的脚本。
- `tools/旧脚本/`：早期启动和打包脚本，正常使用时不用点。

## 不要随便删除

- `data/`
- `uploads/`
- `backups/`
- `app.py`
- `templates/`
- `static/`
- `Windows一键打包.bat`
