# 企业采购仓库销售管理系统 - 打包交付说明

## 一、适合发给别人的方式

当前推荐使用“文件夹打包版”：

```text
管理系统软件/
```

把整个文件夹压缩成 ZIP，发给别人。对方解压后运行启动脚本即可。

## 二、别人怎么启动

### Mac 电脑

双击：

```text
start_mac.command
```

如果系统提示没有权限，可以打开终端执行：

```bash
chmod +x start_mac.command
./start_mac.command
```

然后浏览器打开：

```text
http://127.0.0.1:8876
```

### Windows 电脑

双击：

```text
start_windows.bat
```

然后浏览器打开：

```text
http://127.0.0.1:8876
```

## 三、对方电脑需要什么

需要安装：

```text
Python 3
```

第一次启动时，脚本会自动安装项目依赖：

```text
Flask
openpyxl
```

如果对方电脑不能联网，需要提前把依赖也打包进去，或者改成真正的 exe/app 版本。

## 四、哪些文件必须一起带走

不要只发 `app.py`。建议整个文件夹一起发。

核心文件：

```text
app.py
requirements.txt
templates/
static/
导入示例Excel/
start_mac.command
start_windows.bat
```

数据文件：

```text
data/app.db
uploads/
backups/
```

说明：

- 如果要把已有数据也发给别人，带上 `data/app.db` 和 `uploads/`。
- 如果要给别人一个空白系统，可以删除 `data/app.db` 后再打包；软件启动时会自动生成空数据库。
- `uploads/` 是附件目录，如果不带，图片、PDF、Word、Excel 附件会缺失。

## 五、局域网多人使用

如果对方想让同一个办公室多台电脑访问：

1. 在一台电脑上启动软件。
2. 查看这台电脑的局域网 IP。
3. 其他电脑浏览器访问：

```text
http://这台电脑IP:8876
```

例如：

```text
http://192.168.1.100:8876
```

## 六、以后可以升级成真正安装包

当前文件夹版最容易维护。

后续如果要做成：

```text
Windows exe
Mac app
```

可以使用 PyInstaller 打包。但要注意：

- Windows exe 最好在 Windows 电脑上打包。
- Mac app 最好在 Mac 电脑上打包。
- 打包后体积会更大。
- 每次改代码后需要重新打包。
