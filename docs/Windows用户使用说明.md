# 企业采购仓库销售管理系统 - Windows用户使用说明

## 一、怎么启动

解压收到的压缩包后，双击：

```text
企业采购仓库销售管理系统.exe
```

正常情况下，软件会自动打开浏览器。

如果浏览器没有自动打开，请手动打开浏览器，访问：

```text
http://127.0.0.1:8876
```

## 二、默认账号

管理员：

```text
用户名：admin
密码：admin123
```

普通用户：

```text
用户名：user
密码：user123
```

第一次正式使用后，建议管理员尽快新增自己的账号，并修改默认密码。

## 三、数据保存在哪里

数据保存在 exe 同级目录下：

```text
企业采购仓库销售管理系统数据
```

里面主要有：

```text
data\app.db       数据库
uploads\          上传附件
backups\          备份文件
```

不要随便删除这个文件夹，否则数据会丢失。

## 四、怎么备份

进入软件后，用管理员账号点击：

```text
备份数据库
```

备份文件会保存在：

```text
企业采购仓库销售管理系统数据\backups
```

## 五、能不能断网使用

可以。

这个软件是在本机运行的，不需要互联网。

## 六、常见问题

如果双击 exe 后 Windows 提示安全风险：

1. 点击“更多信息”
2. 点击“仍要运行”

如果打不开：

1. 确认不要把 exe 单独拿出来运行。
2. 尽量保持整个文件夹结构不变。
3. 检查是否被杀毒软件拦截。

## 七、不要做的事

- 不要删除 `企业采购仓库销售管理系统数据`
- 不要只复制 exe 而不复制数据文件夹
- 不要多人同时在不同电脑复制同一份数据各自修改，否则数据会分散
