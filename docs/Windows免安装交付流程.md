# Windows免安装交付流程

## 目标

让对方 Windows 电脑不用安装 Python，解压后双击 exe 就能使用。

## 一、在打包电脑上做一次

准备一台 Windows 电脑。

这台打包电脑需要安装：

```text
Python 3
```

然后把整个项目文件夹复制到 Windows 电脑。

双击：

```text
Windows一键打包.bat
```

等待完成后，会生成：

```text
Windows免安装交付包
```

## 二、发给别人

把这个文件夹压缩成 ZIP：

```text
Windows免安装交付包.zip
```

发给对方。

## 三、对方怎么用

对方解压后，双击：

```text
企业采购仓库销售管理系统.exe
```

不需要安装 Python。

## 四、数据选择

如果要把当前已有数据一起发给对方：

保留交付包里的：

```text
企业采购仓库销售管理系统数据\data\app.db
企业采购仓库销售管理系统数据\uploads
```

如果想给对方一个空白系统：

打包前可以删除项目里的：

```text
data\app.db
uploads\
```

软件第一次启动时会自动生成新的空数据库。

## 五、重要提醒

- 每次修改软件代码后，都要重新运行 `Windows一键打包.bat`。
- 不要只发 exe，建议发整个 `Windows免安装交付包`。
- 数据文件夹不能随便删，里面保存了数据库和附件。
- 如果 Windows 提示“未知发布者”，点击“更多信息”再点击“仍要运行”。
