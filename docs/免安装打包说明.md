# 免安装打包说明

目标：让对方电脑不用安装 Python，双击程序即可使用。

## 一、重要说明

Windows exe 最好在 Windows 电脑上打包。

Mac app 最好在 Mac 电脑上打包。

原因是 PyInstaller 通常不能在 Mac 上直接生成 Windows exe，也不能在 Windows 上直接生成 Mac app。

## 二、Mac 免安装版

在 Mac 上安装一次打包工具：

```bash
python3 -m pip install pyinstaller
```

然后双击：

```text
build_mac_app.command
```

生成位置：

```text
dist/企业采购仓库销售管理系统.app
```

把这个 `.app` 连同 `data/`、`uploads/` 一起发给 Mac 用户。

## 三、Windows 免安装版

Windows exe 必须在 Windows 电脑上打包。打包电脑需要安装 Python，最终使用软件的人不需要安装 Python。

最简单方式：

```text
双击 Windows一键打包.bat
```

脚本会自动安装/检查 PyInstaller，并生成：

```text
Windows免安装交付包
```

把这个文件夹压缩成 ZIP 发给对方即可。

如果想手动打包，也可以在 Windows 电脑上安装一次打包工具：

```bat
py -m pip install pyinstaller
```

然后双击：

```text
build_windows_exe.bat
```

生成位置：

```text
dist\企业采购仓库销售管理系统.exe
```

手动打包时，需要把 `.exe` 连同数据目录一起发给 Windows 用户。

## 四、数据说明

`data/app.db` 是数据库。

`uploads/` 是附件目录。

如果要发空白系统，可以删除 `data/app.db` 后再打包或发送。

如果要带已有数据，必须一起带上：

```text
data/app.db
uploads/
```

## 五、运行方式

用户双击 app 或 exe 后，系统会自动打开浏览器。

如果浏览器没有自动打开，可以手动访问：

```text
http://127.0.0.1:8876
```
