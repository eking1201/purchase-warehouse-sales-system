# 远程试用说明

## 适用场景

你想让别人先打开软件看一看、试一试、提修改意见。

对方不需要安装 Python，也不需要 Windows 电脑打包版。

## 使用方式

在你的 Mac 上双击：

```text
开启远程试用.command
```

如果看到 `https://` 开头的网址，把这个网址发给对方即可。

对方只需要浏览器打开这个网址。

## 第一次使用前

这个方式需要你的 Mac 安装一次 `cloudflared`。

如果脚本提示未检测到 `cloudflared`，请在终端执行：

```bash
brew install cloudflared
```

安装完成后，再双击：

```text
开启远程试用.command
```

## 默认账号

管理员：

```text
admin / admin123
```

普通用户：

```text
user / user123
```

## 重要注意

- 你的 Mac 必须开机。
- `开启远程试用.command` 的窗口不能关闭。
- 电脑睡眠、断网、关闭窗口后，远程网址会失效。
- 这个网址适合临时试用，不适合长期正式使用。
- 对方试用时操作的是你当前这份测试数据库。

## 建议发给对方的话

```text
这是采购仓库销售管理系统的试用链接：

（这里粘贴 https:// 开头的网址）

管理员账号：admin
密码：admin123

你可以先试一下供应商、客户、产品、采购、销售、库存和统计这些模块。
试完后把需要修改的地方发给我。
```

## 如果想长期固定网址

临时网址每次启动可能会变化。

如果后续要长期给别人访问，建议部署到云服务器或 Python 托管平台。
